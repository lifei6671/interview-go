# 链表内指定区间反转

**描述**

将一个节点数为 `size` 链表 `m` 位置到 `n` 位置之间的区间反转，要求时间复杂度`O(n)`，空间复杂度`O(1)`。
例如：

给出的链表为 `1→2→3→4→5→NULL`,`m=2`,`n=4`,

返回 `1→4→3→2→5→NULL`.

数据范围： 链表长度 `0<size≤1000`，`0<m≤n≤size`，链表中每个节点的值满足 `∣val∣≤1000`

要求：时间复杂度 `O(n)` ，空间复杂度 `O(n)`

进阶：时间复杂度 `O(n)`，空间复杂度 `O(1)`

**示例1**

```
输入：{1,2,3,4,5},2,4
返回值：{1,4,3,2,5}
```

**示例2**

```
输入：{5},1,1
返回值：{5}
```

**示例3**

```
输入： {1,2,3,4},1,4
返回值：{4,3,2,1}
```
